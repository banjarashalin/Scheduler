Scheduler

A Web and Mobile application for scheduling and notifying users about their upcoming appointments.

Download this project as a .zip file Download this project as a tar.gz file

Scheduler is an web and mobile(Android) application build for Centennial College.

Background

Very long queue ups in front of Government organizations and places like college enrollment department is a normal scene even in this 21st century, even when people uses computers and internet to do each and everything.

Even after standing in such long queues to schedule an appointment, people has to wait hours to meet the counselor, hoping to call their name to be called out. As students of the technology department of Centennial College, we would like to make proper implementation of technology to solve this problem.

Solution

Scheduler was build as a web application and mobile application containing following functionality's

  1. Appointment Booking.
  2. View Expected Meeting Time.
  3. Sending and Receiving real time notifications on Android smart phones.
  4. Assigning different roles to users.
  5. Getting analytic data on appointments booked cancelled and completed for specific period of time.
  6. Counselors can write notes regarding the meeting which can be viewed by superior authorities.
  7. Counselors and Department Heads can notify a group of students through real-time messaging service.

Technologies

  • Frameworks : Java Spring MVC and Android
  • Languages : Java, XML, JavaScript and PL/SQL
  • Front-end : HTML 5, CSS 3, Bootstrap, JSP, JQuery and Ajax
  • Persistence: MySQL and JSON
